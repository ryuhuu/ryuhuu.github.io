Unicore — Config & Deploy

Estimated read: 5 min

1 — Environment

Example `.env` file:

UNICOREDB_URL=postgres://user:pass@localhost:5432/unicore PORT=8080 NODE_ENV=production

2 — Deploy

Deploy steps (Docker):

docker build -t unicore:latest . docker run -d -p 8080:8080 --env-file .env unicore:latest
Pro tip: add a healthcheck & alerts.
