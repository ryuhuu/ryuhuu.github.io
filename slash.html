1 — Setup

Follow these steps to get SLASH running locally.

git clone https://example.com/slash.git cd slash npm install npm run dev

2 — Common Commands

  • slash start — start local server
  • slash migrate — run DB migrations

3 — Troubleshooting

If you see an error on boot, check the logs above or run with DEBUG=1.
